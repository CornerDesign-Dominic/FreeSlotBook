# Slotly 1.0

Slotly is a calendar-based scheduling application built with Expo, Firebase Auth, Firestore, and Firebase Functions. Read this file first if you want a quick project overview, the main local commands, and the right entry points into the documentation.

## Quick Start

```bash
npm install
npx expo start
```

## Verification

```bash
npx tsc --noEmit
npm run lint
npm run test:firestore
```
